# Music bed license

`bed.mp3` was synthesized in-repo with ffmpeg (pure sine-wave additive pad +
sub-bass pulse; the exact filtergraph is recorded in git history for this file
and in `docs/handoffs/producer-handoff.md`'s build log). It contains no sampled
or third-party material.

To the extent possible under law, the Split Decision project dedicates this
audio to the public domain under **CC0 1.0 Universal**
(https://creativecommons.org/publicdomain/zero/1.0/). Use it for anything,
no attribution required.

Note: the originally planned source, freepd.com, shut down (verified
2026-07-05), so the bed is generated rather than sourced.
